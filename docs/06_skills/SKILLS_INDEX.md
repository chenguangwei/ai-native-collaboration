## 技能触发速查

### 开发工作流

```
需求分析    → /brainstorming
制定计划    → /writing-plans
TDD 开发   → /test-driven-development + /tdd-workflow
代码规范    → /coding-standards (TS) | /java-coding-standards (Java)
架构设计    → /springboot-patterns | /backend-patterns | /api-design
代码审查    → /review → /receiving-code-review
完成验证    → /verification-before-completion
发布流程    → /ship → /land-and-deploy
```

### 质量保障

```
发现 Bug    → /systematic-debugging
根因调查    → /investigate
QA 测试     → /qa | /qa-only
视觉测试    → browser-qa
安全审计    → /cso
性能基线    → /benchmark
```

### Agent 编排

```
独立任务    → /dispatching-parallel-agents
子代理开发  → /subagent-driven-development
自动规划    → /autoplan
Agent 设计  → agent-harness-construction
```
